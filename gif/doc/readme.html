TGIFImage version 2.2 release notes 

Welcome to the TGIFImage version 2.2 release notes.

This eighth release of TGIFImage is the result of more than 20 months of continual research, development and testing. It is a major improvement over previous versions in terms of features, quality, speed and platform support. If you are new to TGIFImage, I suggest you start by reading the description of it at the TGIFImage home page: http://www.melander.dk/delphi/gifimage/.

I estimate that I have spent more than 1000 hours fiddling with this thing and the supporting infrastructure. 1000 hours that I could have spent sleeping, making love, reading, getting drunk and whatnot, but hey I'm not complaining - I did it in Delphi!

Since TGIFImage is freeware, I have no idea how many people are actually using it, but the combined download count from Torry's and melander.dk totals more than 13,000 downloads and of those, close to 2,500 have registered for upgrade notification.

I hope you will have as much fun using TGIFImage, as it has been to develop. Now go read the FAQ.

Enjoy!

Anders Melander
